# YOLO Dataset

This dataset is too large to be stored in GitHub.

Download it here:

https://www.kaggle.com/datasets/pkdarabi/the-drug-name-detection-dataset

Then extract the folders into:
data/yolo/

Folder structure:
- train/images
- train/labels
- valid/images
- valid/labels
- test/images
- test/labels
- data.yaml
